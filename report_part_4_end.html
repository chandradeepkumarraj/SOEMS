CHAPTER 6: CONCLUSION AND FUTURE SCOPE

6.1 Conclusion

The SOEMS project successfully achieves its primary objective of creating a secure, scalable, and user-friendly online examination platform. By leveraging the MERN stack and Socket.io, we have addressed key challenges in remote assessment, particularly regarding integrity and real-time monitoring.

The system effectively differentiates between roles, ensuring data security and streamlined workflows. The implementation of auto-proctoring features significantly reduces the burden on human proctors while maintaining high standards of exam integrity.

6.2 Limitations

  • Requires a stable internet connection for real-time features.
  • Webcam-based proctoring relies on client-side permissions.

6.3 Future Scope

  • AI Integration: Analyzing student facial expressions for stress or suspicious behavior using Machine Learning.
  • Mobile App: Developing a native mobile application for better access on phones.
  • Offline Mode: PWA features to allow syncing answers when connection restores.

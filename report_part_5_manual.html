APPENDIX A: USER MANUAL

A.1 Student Guide

  1. Login: Enter your registered email and password.
  2. Dashboard: View "Upcoming Exams" list.
  3. Start Exam: Click "Start" at the scheduled time. Ensure Camera permission is granted.
  4. Interface:
    • Use "Next/Prev" to navigate questions.
    • The Palette on the right shows answered/unanswered status.
    • Do not switch tabs or exit fullscreen, it will capture a violation.
  5. Submit: Click "Submit Exam" to finish. Result will be displayed immediately.

A.2 Teacher Guide

  1. Create Question: Go to Question Bank -> Add New. Fill details and correct option.
  2. Create Exam: Go to Exams -> Create.
    • Step 1: Basic Info (Title, Duration).
    • Step 2: Add Questions from Bank.
    • Step 3: Set Proctoring Config (High/Medium/Low Security).
    • Step 4: Assign to Groups.
  3. Publish: Click "Publish" to make it visible to students at start time.
  4. Analytics: Open Exam details -> Analytics tab to view histograms and pass rates.

A.3 Admin Guide

  1. User Management: Use "Bulk Import" to upload CSV of students.
  2. Monitoring: Open "Live Monitor" to see active exam sessions and violation alerts in real-time.
